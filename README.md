# VOLTERRA

Premium interactive website for an integrated energy systems company.

## Run locally

```bash
npx serve .
```

Open the local URL shown in the terminal. You may also open `index.html` directly, but a local HTTP server is recommended.

Internet access is required for GSAP and Lenis CDN scripts.

## Structure

- `index.html` — public website markup and brand content
- `styles.css` — visual system, layouts and responsive behavior
- `scripts/app.js` — interactions, energy journey, impact model and motion
- `assets/favicon.svg` — favicon
- `ART_DIRECTION.md` — visual concept
- `CASE_STUDY_RU.md` — Russian case description
- `REELS_SHOT_LIST_RU.md` — recording sequence
- `START_WINDOWS.bat` — Windows launch helper
- `START_MAC_LINUX.sh` — macOS/Linux launch helper

## Status

Original concept project. The public website contains no portfolio or concept labels.
