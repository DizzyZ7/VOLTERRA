# VOLTERRA — описание кейса

VOLTERRA — интерактивный сайт компании, проектирующей комплексные энергетические системы.

Задача проекта — показать энергетику не как набор отдельных электростанций и устройств, а как единый управляемый поток: от генерации и накопления до адаптивной сети и городской инфраструктуры.

Главная сцена строится вокруг энергетического ядра. Далее пользователь проходит четыре состояния системы, взаимодействует с интерфейсом диспетчерской, изучает платформы разного масштаба и меняет параметры модели мощности.

Сайт создан на HTML, CSS и JavaScript. Для scroll-анимаций используются GSAP, ScrollTrigger и Lenis. Визуальные объекты выполнены средствами CSS и SVG без внешних фотографий.
